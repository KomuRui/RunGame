タイトル名 : SUPER_RUN_GAME
制作者     : 小村 琉偉
制作期間   : 2週間

1.操作説明

  Lスティック : 左右移動
  Aボタン     : ジャンプ
  RTボタン    : 回転攻撃

2.仕様

　30秒間で進んだ距離を競い合うゲームです。
　敵や障害物に当たってしまうと,ゲームオーバーです。
　30秒間生き延びた場合,死んだ場合どちらともそれまでに進んだ距離が最後に表示されます。

　Playerは回転攻撃をして敵を倒します。
　敵を倒すと一瞬だけ速度が上がります。
  連続で倒すとコンボが重なり,コンボが重なっている間スピードが上がるので距離が延びます。
  Playerの回転攻撃は時間が経つと,通常の状態に戻るので注意してください。	

　コインを取ると1枚につき,スピードが1.2倍づつ上がっています。

　できるだけコイン取り,敵を倒しながら進むと距離が延びます.
